# 任务插件
共 `110` 款任务插件    
## 1. 证书申请

| 序号 | 名称 | 说明 |
|-----|-----|-----|
| 1.| **证书申请（JS版）** | 免费通配符域名证书申请，支持多个域名打到同一个证书上 | 
| 2.| **证书申请（Lego）** | 支持海量DNS解析提供商，推荐使用，一样的免费通配符域名证书申请，支持多个域名打到同一个证书上 | 
| 3.| **商用证书托管** | 手动上传自定义证书后，自动部署（每次证书有更新，都需要手动上传一次） | 
| 4.| **获取阿里云订阅证书** | 从阿里云拉取订阅模式的商用证书 |
